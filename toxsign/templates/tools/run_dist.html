Signature enrichment analysis tool
The signature enrichment analysis module allows users to compare a selected toxicogenomic signatures to the overall set of signatures deposited by the community using a distance analysis approach (based on the Euclidean distance and the correlation coefficient).
How to perform enrichment
Input data: Select the signature using the "Select signature" selector
Output data: Table that corresponds to the distance between the selected signature and all other TOXsIgN public signatures (including your own private signatures), where N is the total number of HomoleGene Ids measured, R is the total number of HomoleGene Ids meeting the criterion, n is the total number of HomoleGene Ids in this specific signature, and r is the number of HomoleGene Ids meeting the criterion in this specific signature and ratio corresponding to r/union(n,R).
